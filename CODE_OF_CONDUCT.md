# Code of Conduct

We are committed to providing a friendly, safe, and welcoming environment for all, regardless of level of experience, gender identity, sexual orientation, disability, personal appearance, body size, race, ethnicity, age, religion, or nationality.

## Expected Behavior
- Be kind and respectful.
- Assume good intent.
- Give and accept constructive feedback.

## Unacceptable Behavior
- Harassment or discrimination in any form.
- Personal attacks, trolling, or sustained disruption.

## Enforcement
Project maintainers are responsible for clarifying standards and applying them fairly. Report issues to the maintainers via the repository issue tracker.
